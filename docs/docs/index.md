# TERRA AAI

This is the documentation site for the TERRA AAI. The service is part of the wider [TERRA](https://terra-horizon.eu/) platform. This is part of the overall [platform documentation](https://terra-horizon.github.io/).

The platform's AAI service acts as the horizontal Authentication and Authorization Infrastructure solution for TERRA user interface and all other core or integrating components that want to utilize the TERRA offerings. 

You can use the menu options on the left to navigate through the available documentation. You may be interested to jump directly to an [Architecture Overview](architecture.md), see the available [Onboarding Material](onboarding.md) or see some token request example in the API section. You can find here the service [code repository](https://github.com/terra-horizon/terra-aai).

For any questions, you can consult the [FAQ](faq.md), check if there is a relevant [issue](https://github.com/terra-horizon/terra-aai/issues) answering your question or contact the [TERRA Help Desk](https://terra-horizon.eu/contact-us/).
